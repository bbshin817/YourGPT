# YourGPT
事前に記述した文章を、まるでAIがレスポンシブしているかのように表示させることが出来る、簡易的なWeb APPです。
## 使い方
同梱の**msgs.json**へ、配列形式でメッセージを記述します。
YourGPTは、与えられているメッセージを上から順番に出力します。
`msgs.json`
```json
[
	"こんにちは、何かご用ですか？",
	"私にできるのは、あらかじめ入力されたメッセージを出力することだけです。"
]
```
